# web-search

内部知識(学習データ)で即答せず、`WebSearch` / `WebFetch` で一次情報を取りに行ってから回答させる Claude Code skill。

## いつ発動するか

「揺らぐ事実」を聞かれた時に自動発動する。具体的には:

- ライブラリ / SDK / フレームワークの最新バージョン、API シグネチャ、breaking change
- LLM・クラウド・SaaS の料金、モデルID、コンテキスト長、レート制限
- 製品・サービスの現在の状態、ニュース、リリース、人事
- 統計、価格、為替、ランキング
- 「最新」「現在」「今」「今年」「2025」「2026」「最近」など時制キーワードを含む質問

純粋なアルゴリズム説明や時間で変わらない一般論は対象外。

## 何をするか

skill が発動すると Claude は次の手順に縛られる:

1. **まず `WebSearch` を呼ぶ** — 内部知識から書き始めない
2. **公式ソースを `WebFetch`** — 数値・API シグネチャ・料金は一次情報まで踏み込む
3. **食い違いがあれば追加検索** — 一次情報を優先、記事の日付を見る
4. **回答に出典 URL を併記** — 該当箇所の近くに置く
5. **取れなかったことは取れなかったと言う** — 憶測で埋めない

## 検証

`empirical-prompt-tuning` skill で 2 シナリオ (Next.js 最新版 / Claude vs GPT-4o 料金比較) を新規 subagent で実証評価し、[critical] 要件全達成 (精度 100%) を確認済み。
